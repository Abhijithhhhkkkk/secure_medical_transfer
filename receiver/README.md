
# Secure Medical Image Receiver

## Description
This project is the **receiver module** of a secure medical image transmission system.  
It receives encrypted image data from a sender device, decrypts it using hybrid cryptography (RSA + ASCON), verifies its authenticity, stores the image, and displays it on a web interface.

---

## Features
- Secure encrypted packet reception
- RSA private key decryption
- ASCON image decryption
- Authentication tag verification
- Automatic image storage
- Web dashboard image display
